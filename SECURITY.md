# Security Policy

## Supported versions

This repo mirrors the current `main` branch of the [MJSFX](https://mjsfx.app) app. Only the
latest published state is supported; there are no maintained older versions.

## Reporting a vulnerability

This repo holds generated reference data (controller configs, generator recipes, themes,
example sounds and MIDI profiles) published read-only from the MJSFX source tree — see the
README. It has no build step and runs no code, but if you find a vulnerability here (for
example, a file that could be used to exploit something parsing it) or in the MJSFX app itself,
please report it privately using GitHub's **Report a vulnerability** button on this repo's
[Security tab](https://github.com/matthewjschultz/mjsfx-reference/security/advisories/new),
or via [mjsfx.app/support](https://mjsfx.app/support).

Please do not open a public issue for security reports.
